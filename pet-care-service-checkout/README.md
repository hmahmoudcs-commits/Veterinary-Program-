# Pet Care Service Checkout

A C console program for a pet-care or veterinary service checkout system. The program loads pet records and service options from text files, lets the user find or create a customer pet profile, select services, enter payment details, and print a receipt.

## Features
- Loads pet and service data from files
- Searches existing pet/customer records
- Adds new pet/customer entries
- Allows multiple service selections
- Calculates and prints a checkout receipt

## Language
C
